let

Let usamos para salvar um valor na mémoria, com o let você pode mudar ex: let name = "Eduardo" / name = "José"

const

Const você também Salva valor na memoria, porém você não pode alterar igual fazemos com let

Math

Uma função do js Usamos para funçãoes matematica ex: console.log(Math.max(1, 4, 5, 6)) - Max ele vai retornar o numero maior Seria o 6


Floor ele vai redonda o numero para baixo ex: console.log(Math.max(5.16)) - Ficaria 5


Temos o Ceil Que ele irar redonda para cima ex: console.log(Math.ceil(5.14)) - ficaria 6
